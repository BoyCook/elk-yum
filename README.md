# elk-yum
Setup ELK yum installation packages

### command

  sudo ./setup.sh
  beatname -c beatname.yml -configtest

### Services

Services run on host but ports are forwarded from guest to host

* Elastic: http://localhost:9200
* Kiabana: http://localhost:5601
